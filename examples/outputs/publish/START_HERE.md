# HomoRepeat Run: example

## Open These First

- `calls/repeat_calls.tsv`: detected amino-acid repeat tracts.
- `tables/accession_status.tsv`: whether each accession completed, failed, or completed with no calls.
- `tables/accession_call_counts.tsv`: number of calls per accession, method, and residue.
- `summaries/status_summary.json`: run-level counts.
- `metadata/nextflow/report.html`: runtime diagnostics in a real run.

This example is a small static snippet. Real runs include more metadata under
`metadata/`.
